# Flying Frogs

Welcome to Flying Frogs. This project is in early active development. The app is currently unhosted but can be run locally with the following procedure

# Running the app

Clone the repository
```git clone https://github.com/caj00017/FlyingFrogs.git```

Ensure packages are installed
```npm install```

Start the app
```npm run start```

Alternatively, run for development with nodemon
```npm run dev```
